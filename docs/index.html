What is jq-repeat?

A simple, yet highly customizable jQuery plugin to handle all of your client-side repetitive DOM needs. Simple, quick and powerful templating using Mustache syntax. Modeled after ng-repeat with automatic DOM synchronization.

Demo 1: Basic Todo List

Click "Add Task" to see items automatically rendered in the DOM. Use the buttons below to interact with the list.

  • {{ task }}
View Code 
<ul>
    <li jq-repeat="todos">
        <span>{{ task }}</span>
        <button onclick="$(this).scopeItemRemove()">Remove</button>
    </li>
</ul>
// Add items to the list
$.scope.todos.push({ task: 'Buy groceries' });
$.scope.todos.push({ task: 'Walk the dog' });

// Items are automatically rendered!

Demo 2: Add Automatic Sorting

Now let's add sorting to our todo list! Click "Enable Sorting" to add the jr-order-by attribute. New items will be inserted alphabetically.

Try adding tasks after enabling sorting - they'll appear in alphabetical order!

View Code 
<!-- Just add jr-order-by attribute -->
<li jq-repeat="todos" jr-order-by="task">
    <span>{{ task }}</span>
    <button onclick="$(this).scopeItemRemove()">Remove</button>
</li>
// Items are now automatically inserted in sorted order!
$.scope.todos.push({ task: 'Zebra task' });
$.scope.todos.push({ task: 'Apple task' });
// 'Apple task' will appear before 'Zebra task'

Demo 3: Add Custom Animations

Use hooks like __put and __take to add smooth animations when items are added or removed.

Try adding and removing tasks after enabling animations - watch them fade in and out!

View Code 
// Fade in when items are added
$.scope.todos.__put = function($el, item, list) {
    $el.hide().fadeIn(300);
};

// Fade out when items are removed
$.scope.todos.__take = function($el, item, list) {
    $el.fadeOut(300, function() {
        $(this).remove();
    });
};
